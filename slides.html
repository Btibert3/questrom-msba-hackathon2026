Tools

Step 1 of 3

pipeline_flow.py

Pull yesterday's MLB pitches → MotherDuck

No API key needed. The MLB Stats API is free and public. We get every pitch thrown yesterday — pitch type, velocity, zone, count, runners on base, score, and at-bat result.

pipeline_flow.py

Three Tasks

1

get_game_ids — calls statsapi.schedule(date) and returns a list of game IDs for yesterday

2

fetch_pitches — for each game, walks every at-bat and pitch event; extracts 20+ fields per pitch into a flat DataFrame

3

load_to_motherduck — connects to md:?motherduck_token=…, creates hackathon.pitches with CREATE OR REPLACE TABLE

@flow(name="pipeline-flow")
def pipeline_flow():
    yesterday = (date.today() - timedelta(days=1)).strftime("%Y-%m-%d")
    game_ids = get_game_ids(yesterday)
    df = fetch_pitches(game_ids, yesterday)
    load_to_motherduck(df)

Step 2 of 3

llm_flow.py

Serverless GPU inference on Modal

The model (SmolLM2) is deployed on Modal. First call cold-starts the GPU (~60s). Subsequent calls are fast. The API is OpenAI-compatible — any client works.

llm_flow.py

One Task, One HTTP Call

@task
def call_modal_endpoint(prompt: str) -> str:
    url = os.environ["MODAL_CHAT_URL"]
    payload = {"messages": [{"role": "user", "content": prompt}]}
    response = httpx.post(url, json=payload, timeout=120)
    text = response.json()["choices"][0]["message"]["content"]
    print(text)
    return text

@flow(name="llm-flow")
def llm_flow(prompt: str = "In one sentence, what makes a slider an effective strikeout pitch?"):
    call_modal_endpoint(prompt)

Any Prefect flow can call this — it's just a POST to an HTTPS endpoint.

Step 3 of 3

langchain_flow.py

LangChain chain + LangSmith observability

Query MotherDuck for a pitch summary, feed it to a LangChain chain backed by our Modal LLM, and ask for an insight. Every call is traced automatically in LangSmith.

langchain_flow.py

Chain = Prompt | Model

# Custom LangChain model wrapping our Modal endpoint
class ModalChatModel(BaseChatModel):
    endpoint_url: str
    def _generate(self, messages, ...):
        # POST to Modal, return AIMessage

@task
def run_chain(context: str) -> str:
    llm = ModalChatModel(endpoint_url=os.environ["MODAL_CHAT_URL"])
    prompt = ChatPromptTemplate.from_messages([
        ("system", "You are a data analyst."),
        ("human", "Here is yesterday's MLB pitch data:\n\n{context}\n\n"
                  "In 2-3 sentences, what is the most interesting insight?"),
    ])
    chain = prompt | llm
    result = chain.invoke({"context": context})
    return result.content
